# Weather App
## Author: Bilal Haider

### Setup:
- Download git repo
- go to root of project
- install pod using
```
pod install
```

### Run:
- open xCode
- connect iPhone with xCode
- run application

### Project Info:
it uses MVVM
- Model: Model objects i.e WeatherData
- View: View classes: ContentView, WeatherView
- View Model: Holds business logic i.e WeatherViewModel
